# Project Skills

Each subdirectory in `project/skills/` is a project-oriented installable skill.

Current skills:

- `execplan-grill`: Use as the first step when the user wants an ExecPlan.
- `execplan-review`: Use when the user requests a review of the ExecPlan.
- `grill-me`: Stress-test a plan or design through a one-question-at-a-time interview. Subtree-managed from Matt Pocock's [`grill-me`](https://github.com/mattpocock/skills/tree/main/skills/productivity/grill-me) skill.
- `humanizer`: Remove signs of AI-generated writing from text. Subtree-managed from [`blader/humanizer`](https://github.com/blader/humanizer).
- `precommit-review`: Use when the user requests a phased pre-commit review of the worktree.
- `reference-module`: Use to research git repos when user requests.
- `chatgpt-pro-review`: Use when the user explicitly asks Codex to consult ChatGPT Pro through the Codex app browser.
- `ui-review-claude`: Use to review UI screenshots in parallel with the installed Claude Code CLI.
