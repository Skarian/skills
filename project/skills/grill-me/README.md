# grill-me

Project-level copy of Matt Pocock's `grill-me` skill for stress-testing a plan or design through a focused interview.

Copied verbatim from Matt Pocock's [`grill-me`](https://github.com/mattpocock/skills/blob/main/skills/productivity/grill-me/SKILL.md) skill.

Behavior:

- Interviews one question at a time until the plan is clear.
- Walks the decision tree and resolves dependencies between decisions.
- Provides a recommended answer with each question.
- Explores the codebase instead of asking when the answer is discoverable.

Install (Codex):

```bash
npx skills add https://github.com/Skarian/codex-skills/tree/main/project --skill grill-me -a codex -y
```
